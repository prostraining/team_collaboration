PROS Platform for Airlines

Accelerated Value Revenue: Management Essentials 102 - Configuration Workflow

NEW COURSE: Accelerated Value Revenue: Management Essentials 102 - Configuration Workflow was developed for our Accelerated Value customers and will enable users to review the fundamental concepts of RM, familiarize themselves with the interface and know how to input data necessary for system configuration, locate and use reference data, enter and edit post-departure data, and manage users and their roles, work teams, and privileges in the system.

Accelerated Value: Revenue Management Essentials 102 - Optimizer Workflow

NEW COURSE: Accelerated Value: Revenue Management Essentials 102 - Optimizer Workflow was developed specifically for our Accelerated Value customers enabling users to define Bid Prices and how Class Authorization Levels are calculated and how to identify the importance of how availability is used with daily fares through Capacity Management and Optimization.

Accelerated Value: Revenue Management Essentials 102 - Forecast Workflow

NEW COURSE: Accelerated Value: Revenue Management Essentials 102 - Forecast Workflow was developed specifically for our Accelerated Value customers enabling users to identify each Forecast Dimension and explore the importance of how DCPs, Drift, Order of Application of Influences, HSE Alerts and FSS are utilized within the Forecaster Module.

RMA/RME/RME+ 101
Forecast Module

NEW COURSE: [RMA] / [RME] / [RME+ 101] Forecast Module enables users to recognize the value the PROS Forecast Module brings to airline's abilities to forecast demand. Users will be able to identify five critical concepts of the Forecast Module, its four main characteristics and recognize the different dimensions within each of the four forecast modules.
